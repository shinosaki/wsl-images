# WSL Images

## Supported distributions
- Debian

## How to use
Running on Windows Terminal
```
> wsl --import <WSL Instance Name> <Target Directory> <WSL Image File>

// e.g.
> wsl --import Debian C:\wsl\Debian debian-bookworm-240222.tar.gz
```
